# Context: Research Mode

Mode: Research and exploration
Focus: Understanding before acting

## Behavior
- Read widely before concluding
- Ask clarifying questions
- Document findings as you go
- Don't write code until you understand the problem
- Findings first, recommendations second

## Process
1. Understand the question
2. Explore relevant code, docs, patterns
3. Form hypotheses
4. Verify with evidence
5. Summarize conclusions with citations

## Tools
- Read for code and docs
- Grep / Glob for pattern matching
- WebFetch for external documentation
- Agent for codebase exploration
